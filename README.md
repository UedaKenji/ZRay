# ZRay

A ray tracing library for optical systems in axisymmetric vessels.

## Development (uv)

```bash
uv sync --dev
```

## Build package

```bash
uv build
```

## Install locally

```bash
uv pip install .
```

## Notes

- This project is a library package, so `uv.lock` is not required by default.
- The bundled DXF sample (`zray/dxf_files/rt1_simple.dxf`) is included in package data.
